## 페어리테일 · 티팟 — 캐릭터 제작 페이지 (가장 공들인 영역)

수십 장의 이미지를 다루는 제작자가 *자기 방식대로* 정리하도록 설계.

- **두 가지 보관 타입** — 폴더 / 일반
- **세 가지 뷰** — List · Grid · Split
- **Drag & Drop** — *폴더 합치기 / 이미지 합치기 / 폴더 해제*

그리고 작성 중 사고를 막는 흐름:

- **체크포인트 기반 버전 관리** (최대 50개) + **10분 주기 자동저장**
- 등록 완료된 CP는 덮어쓰기 금지 — **새 체크포인트 생성**
- 작성 도중 새로고침·뒤로가기 시 **이탈 가드 다이얼로그**
- detail 모드 — **Jinja2(nunjucks)** 템플릿 + 30,000자 진행률 측정

같은 기능을 유지하면서 정보 밀도·동선을 다시 짠 리뉴얼.

- 친구 기능(Best/Worst Friends/매칭 기록) — **단일 재사용 컴포넌트**로 모두 동일한 UX 보장
