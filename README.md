# about-me

이선위 · Frontend Developer 포트폴리오

- **배포 URL:** https://ricklee1998.github.io/about-me/
- **스택:** React · TypeScript · Vite · Tailwind CSS

## 로컬 개발

```bash
npm install
npm run dev
```

로컬에서는 `http://localhost:5173/about-me/` 로 접속합니다.

## 빌드

```bash
npm run build
npm run preview
```

## GitHub Pages

`main` 브랜치에 push하면 `.github/workflows/deploy.yml` 이 자동으로 빌드·배포합니다.

Repository Settings → Pages → Source: **GitHub Actions** 로 설정되어 있어야 합니다.
